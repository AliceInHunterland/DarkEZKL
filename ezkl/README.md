# Embedded EZKL

This `ezkl/` directory is the bundled proving engine used by the root Dark-EZKL benchmark workflow.

If you want to run the benchmark, start at:

- [../README.md](../README.md)

If you want the Dark-EZKL probabilistic design notes, see:

- [../PROBABILISTIC.md](../PROBABILISTIC.md)

If you need offline trusted setup reconstruction, see:

- [../SRS_RECONSTRUCTION.md](../SRS_RECONSTRUCTION.md)

The older benchmark-oriented content that used to live in this file referenced removed repo paths such as `BENCHMARKS.md`, `benchmark.py`, `bench_vit.py`, and `docker-compose.yml`. It was intentionally replaced with this shorter pointer so there is a single current runbook.

The embedded EZKL security notes that still apply live under:

- [docs/advanced_security/public_commitments.md](docs/advanced_security/public_commitments.md)
- [docs/advanced_security/quantization_backdoors.md](docs/advanced_security/quantization_backdoors.md)
